Quickstart

Get started in
3 lines of code

Install, wrap your agent, chat. The evolution happens automatically behind the scenes.

1
Install — one npm package
2
Wrap — give your agent a genome
3
Chat — it learns and evolves
my-agent.ts
import { PGA } from '@gsep/core';

// Give your agent a genome
const agent = PGA.wrap('my-assistant', {
  provider: 'anthropic',
  model: 'claude-sonnet-4-5-20250929',
  preset: 'standard',
});

// Chat — it evolves automatically
const reply = await agent.chat(
  'Help me design an API',
  { userId: 'user-42' }
);

// Prompts mutate, compete, improve
// Per-user adaptation happens automatically
// Bad mutations roll back, good ones propagate
