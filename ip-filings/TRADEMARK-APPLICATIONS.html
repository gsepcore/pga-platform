Filing Process (USPTO)

Option A: File Online via TEAS

URL: https://www.uspto.gov/trademarks/apply

Steps: 1. Create USPTO account 2. Select TEAS Plus or TEAS Standard: - TEAS Plus: $250/class (stricter requirements) - TEAS Standard: $350/class (more flexible) 3. Fill out application form 4. Upload specimens 5. Pay filing fees 6. Submit application 7. Receive serial number immediately

Timeline: - Filing to First Office Action: 3-6 months - Total to registration: 8-12 months (if no objections)

USING YOUR TRADEMARKS

Before Registration

You CAN use: - ™ symbol (common law trademark) - Example: “PGA™ Platform” - No registration needed

You CANNOT use: - ® symbol (reserved for registered marks) - Using ® before approval is illegal and can jeopardize application

After Registration

You MUST use: - ® symbol (shows registration) - Example: “PGA® Platform”

Best Practices: - Use symbols consistently in all materials - Police unauthorized use by others - Maintain quality standards (trademarks can be canceled for non-use) - Keep evidence of use for renewals
